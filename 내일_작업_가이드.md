## 4. 자주 쓰는 git 명령어 치트시트

| 하고 싶은 것 | 명령어 |
|---|---|
| 뭐가 바뀌었나 확인 | `git status` |
| 바뀐 내용 자세히 보기 | `git diff` |
| 커밋할 파일 담기 (전체) | `git add -A` |
| 로컬에 기록 | `git commit -m "메시지"` |
| GitHub로 발송 | `git push` |
| 최근 기록 보기 | `git log --oneline -10` |
| 지금 어느 브랜치? | `git branch` |

> 💡 순서 외우기: **status → add → commit → push** (확인 → 담기 → 박제 → 발송)
