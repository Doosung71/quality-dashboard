## 검수 요청 항목

### A. 라벨 동기화 완전성 — 누락된 항목이 없는가?
**위치**: `MainDashboard.tsx` L700–L790 퀵 링크 위젯 전체
**내용**: 사이드바(`sidebar.tsx`) 의 모든 1·2depth 메뉴명과 퀵 링크 내 라벨을 대조해 위 4개 항목 외에 불일치가 없는지 확인
**리스크**: 사이드바에서 변경됐지만 퀵 링크에서 놓친 항목이 있으면 사용자 혼란 유발

### B. 링크 경로(href)와 라벨 정합성
**위치**: `MainDashboard.tsx` L768–L788 (도구 그리드 4개 링크)
**내용**: `입찰 프로젝트` 라벨이 `href="/dashboard"` 를 유지하는 것이 적절한지 검토
**리스크**: 메뉴명 `입찰 프로젝트`와 실제 라우트 `/dashboard`가 의미적으로 괴리될 수 있음

### C. 권한 맵 위젯 내 구 메뉴명 잔존 여부
**위치**: `MainDashboard.tsx` L293–L303 (권한 맵 `label` 배열)
**내용**: 관리자 전용 "섹션별 접근 권한" 칩에 `공급망관리`, `시험장`, `QKM` 등 구 명칭이 하드코딩되어 있음 — 이것도 이번에 함께 갱신해야 하는가, 아니면 별도 작업인가
**리스크**: 권한 맵만 구 명칭으로 남으면 관리자 혼란 유발. 단, 스코프 확장에 해당하므로 Dennis 결정 필요
